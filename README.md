# Prérequis

Vous devez vous assurer les différents elements suivants sont installés sur votre machine.

* PHP >= 7
* OpenSSL PHP Extension
* PDO PHP Extension
* Mbstring PHP Extension
* XML PHP Extension
* JSON PHP Extension
